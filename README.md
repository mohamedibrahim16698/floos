# floos

بلان مبدئي لتطبيق تتبّع الفلوس:

## 1) إدارة الحسابات والربط مع الرسائل
- إضافة كل الحسابات (Debit / Credit / Wallet / Bank Account).
- لكل حساب: اسم، نوع، عملة، رصيد ابتدائي، وإمكانية تعطيله.
- ربط كل بطاقة بآخر 4 أرقام (مثال: `1234`) لاكتشاف العمليات تلقائيًا من الرسائل.
- دعم ربط حسابات التحويل (زي InstaPay) بآخر 4 إلى 6 أرقام من رقم الحساب.
- خيار “استبعاد من إجمالي الرصيد” للحسابات اللي مش عايزها تدخل في الإجمالي.

## 2) الاشتراكات والأقساط
- شاشة للاشتراكات الدورية: (اسم الاشتراك، القيمة، تاريخ الاستحقاق، التكرار، الحساب المرتبط).
- شاشة للأقساط: (إجمالي القسط، المدفوع، المتبقي، موعد القسط القادم).
- تنبيهات قبل موعد الدفع.

## 3) Budget & Goals
- Budgets شهرية حسب الكاتيجوري (أكل، مواصلات، تسوق...).
- Goals ادخار (هدف، مبلغ مستهدف، تاريخ مستهدف، نسبة الإنجاز).
- تنبيهات عند تخطي الميزانية أو الاقتراب من الموعد المستهدف.

## 4) Insights
- ملخص شهري: دخل / مصروف / صافي.
- أعلى كاتيجوري صرف.
- مقارنة شهر بشهر.
- متوسط الصرف اليومي والأسبوعي.

## 5) التتبّع التلقائي لرسائل البنوك
- قراءة رسائل الـSMS (بموافقة المستخدم فقط).
- استخراج: المبلغ، نوع العملية، الوقت، آخر أرقام البطاقة/الحساب، الرصيد بعد العملية (لو متاح).
- مطابقة الرسالة بالحساب الصحيح باستخدام:
  - آخر 4 أرقام البطاقة
  - آخر 4-6 أرقام الحساب للتحويلات
- حفظ العملية تلقائيًا في التطبيق مع كاتيجوري مبدئي.

### كلمات مفتاحية مبدئية للاكتشاف
`transaction`, `purchase`, `payment`, `debit`, `credit`, `spent`, `balance`, `withdraw`, `deposit`, `transfer`,
`عملية`, `خصم`, `اضافة`, `دفع`, `شراء`, `ايداع`, `سحب`, `رصيد`, `تحويل`

## اقتراحات إضافية مفيدة
- Rules ذكية للتصنيف التلقائي (مثلاً أي عملية فيها “Uber” -> مواصلات).
- إمكانية تعديل التصنيف مرة واحدة والتطبيق يتعلم النمط.
- تصدير CSV / Excel للتقارير.
- Backup & Restore (محلي أو سحابي).
- حماية التطبيق ببصمة/Face ID + تشفير البيانات محليًا.
- دعم العملات المتعددة.
